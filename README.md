# Mumper - 2D Engine

Mumper is a 2D Engine written in Rust using egui</br>

## Features

-World Positions in meter</br>
-Polygon Drawing</br>

## Main Goals

-2D Coordinate / Camera system</br>
-Learn 2D Physics from scratch</br>
